# trello_dnd

[![Build Status](https://github.com/Sofa-RV/trello_dnd/workflows/Deploy%20to%20GitHub%20Pages/badge.svg)](https://github.com/Sofa-RV/trello_dnd/actions)

Проект Trello‑стиля с Drag‑and‑Drop.

---

## Как запустить локально

1. Установите зависимости:
   ```bash
   yarn install
   ```

2. Запустите сервер разработки:
   ```bash
   yarn dev
   ```

3. Соберите проект для продакшена:
   ```bash
   yarn build
   ```

Сайт будет задеплоен на GitHub Pages по адресу:  
`https://Sofa-RV.github.io/trello_dnd/`
